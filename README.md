# Ibtikar Smart Commerce

منصة تجارة إلكترونية وتداول ذكية تعمل بالذكاء الصناعي.

## الميزات:
- تسجيل دخول المشرف
- لوحة تحكم للطلبات والصفقات
- ربط مباشر مع Binance API
- تخزين بيانات Firebase (Firestore + Auth)

## البدء:
```bash
npm install
npm start
```

## البناء:
```bash
npm run build
```

## الاستضافة المقترحة:

- Vercel
